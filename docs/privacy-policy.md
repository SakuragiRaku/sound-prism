# プライバシーポリシー (Privacy Policy)

**Sound Prism**（以下、「本拡張機能」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めています。本プライバシーポリシーでは、本拡張機能がどのようにデータを取り扱うかについて説明します。

## 1. 取得する情報
本拡張機能は、音楽のビジュアライザー機能を提供するため、Chromeの `tabCapture` API を使用して、現在アクティブなタブ（主にYouTube）で再生されている音声データを取得します。

## 2. 情報の利用目的
取得した音声データは、以下の目的でのみ使用されます。
- 音声の周波数解析およびキャンバス上へのリアルタイムなビジュアライザー（波形・バー等）の描画。

## 3. データの保存および外部送信について
**本拡張機能は、取得した音声データをユーザーの端末（ローカル）でのみ処理します。**
- 外部のサーバーへのデータの送信は一切行いません。
- 音声データを録音、保存、または収集することは一切ありません。
- ユーザーの個人情報やブラウジング履歴などを収集・送信することもありません。

## 4. 設定データの保存
ユーザーがポップアップメニュー内で変更した設定（ビジュアルモード、カラーテーマ、感度、不透明度などのカスタマイズ情報）は、Chromeのローカルストレージ（`chrome.storage.local`）にのみ保存され、外部に送信されることはありません。

## 5. 同意
本拡張機能をインストールし利用することにより、ユーザーは本プライバシーポリシーに同意したものとみなされます。
