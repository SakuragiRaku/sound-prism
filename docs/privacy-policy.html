Sound Prism

プライバシーポリシー

最終更新日: 2026年4月17日

はじめに

Sound Prism（以下、「本拡張機能」）は、ユーザーのプライバシーを第一に考え、個人情報の保護に努めています。本プライバシーポリシーでは、本拡張機能がどのようにデータを取り扱うかについて透明性をもって説明します。

取得する情報

本拡張機能は、音楽のビジュアライザー機能を提供するため、Chromeの Web API (`tabCapture`) を使用して、現在アクティブなタブ（主にYouTube等のメディアプレイヤー）で再生されている音声データを取得します。

情報の利用目的

取得した音声データは、以下の目的でのみ使用されます。

  • 音声の周波数解析およびキャンバス上へのリアルタイムなビジュアルエフェクト描画。

データの保存および外部送信について

本拡張機能は、取得した音声データをユーザーの端末（ローカル環境）でのみ安全に処理します。

  • 外部のサーバーや第三者へのデータの送信は一切行いません。
  • 音声データを録音、保存、または収集することは一切ありません。
  • ユーザーの個人情報やブラウジング履歴などを収集・送信することもありません。

設定データの保存

ユーザーがメニュー内でカスタマイズした設定（ビジュアルモード、カラーテーマ、感度、不透明度など）は、Chromeのローカルストレージにのみ保存されます。これらの設定データが端末外部に送信されることはありません。

同意

本拡張機能をインストールしご利用いただくことにより、ユーザーは本プライバシーポリシーに同意したものとみなされます。ポリシーの内容は必要に応じて見直され、拡張機能のアップデートを通じて最新版が適用されます。
